# Regional Anaesthesia Notes

A searchable reference for common surgical procedures and their recommended regional anaesthetic techniques.

## How to use this guide

Use the **search bar** at the top or the **sidebar** to find a procedure by name or body region.

Each procedure page includes:

- **Surgical overview** — what the procedure involves
- **Recommended regional technique(s)** — with alternatives where appropriate
- **Key anatomy** — relevant landmarks and sonoanatomy
- **Block-specific considerations** — local anaesthetic choice, volume, catheters
- **References** — links to primary evidence and guidelines
